## Overview
Web application about pokemon.

## Main technologies:
- Python 3.9.4
- Django 3.2.6
- PostgreSQL

### Installation/Setup
- Create a virtual environment
- Install packages using 
```sh
pip install -r requirements.txt
```
- Create database (and set it up in settings.py), then makemigrations and migrate
